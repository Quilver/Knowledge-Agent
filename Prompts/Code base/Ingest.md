## Ingest

When the user says "ingest [source]":

1. Read the source 
2. Create a summary of the source with CreateFile tool
3. Identify all interfaces, functions, and datatypes
4. Record all interface, functions, and datatypes identified if there is not already a page created
5. Add cross-links in both directions between all touched pages
6. Add any new tags
7. Flag any contradictions with existing wiki content
